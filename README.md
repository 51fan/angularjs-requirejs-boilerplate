## Nodejs Basicer
  Completely, basic angularjs frontend web framework for [node](http://nodejs.org).

## Quick Start
  Install dependencies:

```bash
$ npm install
$ bower install
```

  Compile and build system:

```bash
$ gulp
```

  Start the server:

```bash
$ node app
```

  Open the browser and input http://localhost:4250/
